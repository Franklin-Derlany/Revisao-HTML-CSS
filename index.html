Em HTML algumas regrinhas importantes devem ser levadas em consideração.

LISTAS:

É preferível que as listas não estejam dentro de parágrafos, caso seja nescessário colocar uma class nos paragrafos as listas que estiverem dentro dos paragrafos não sofreram o efeito das class.
Ou seja, Escreva tudo dentro da tag <P> e caso precise acrescentar uma lista use a tag <ol> ou <ul> fora do parágrafo, como no exemplo a baixo:
